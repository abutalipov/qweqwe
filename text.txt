Павел, вот один метод высчитывания веса для моего голоса. 
Предположим что в нашей базе данных N человек со скиллом С. 
Каждый из этих людей обладает рейтингом по этому скиллу - рейтинг это количество голосов отданных человеку по этому скиллу. 
Соответственно у нас N рейтингов по скиллу S: R1, R2,..., Rn. 
Высчтиваем среднее арифметическое: Rm = (R1+R2+...+Rn)/N. 
Потом высчитываем "cреднеквадратическое отклонение: S= √{[(R1-Rm)^2+(R2-Rm)^2 + ... + (Rn-Rm)^2]/(N-1)}. 
Вес голоса человека I по скиллу С: Wi,c = (Ri - Rm)/S.

Да, это еще не всё. Результат этих вычислений ограничивается от -3 до 3. 
То есть если мы получили вес -4, мы его назовём -3. 
Потом надо перенести эти результаты в разброс от 0.5 до 3.5 используя слудующую формулу: 0.5 + 1/2 * (Wis + 3).

Формула относительно простая. Сначала мы высчитваем вес скилла по сайту (или по срезу). 
Вес скилла это всего лишь суммарное количество голосов по скиллу по всему сайту (или по срезу) поделенное на суммарное количество голосов по всем скилла по всему сайту (срезу). 
Тоесть это отсноительная популярность скилла. 
Чтобы высчитать единый рейтинг человека, мы умножаем его рейтинг по скиллу С на вес скилла С по сайту (или по срезу) и 
складываем эти "взвешенный рейтинги" вместе чтобы получить единый рейтинг. 
Единый Рейтинг = Популярность Скилла 1 x Рейтинг человека по Скиллу 1 + Популярность Скилла 2 x Рейтинг человека по Скиллу 2 +... +... SПопулярность Скилла N x Рейтинг человека по Скиллу N
